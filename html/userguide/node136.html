Introduction

As discussed in Chapter 3, there are two basic suites of display software for Miriad data. One uses the ATNF Visualisation suite, and the other is PGPLOT-based.

The ATNF visualisation software, which lies outside the domain of Miriad, uses an X-windows based user interface, with pop-up menus, etc. It is largely interactive, and tends to be less useful when quantitative information is needed. The PGPLOT-based software uses the normal Miriad user interface, tends to be better suited to hardcopy output, quantitative work, and for more specialised astronomical applications (e.g. plotting polarisation vectors).
