Other Flags and Arguments

-a
This causes flint to include a crude cross-reference map in its log file.
-I
This flag is used to give the name of a directory to search for include files. When it encounters an `include' statement, flint first checks the local directory for the include file, and then checks other directories given by the -I switch, in the order they were present on the command line. This is also equivalent to -i.
-?
This causes some help information to be printed.
